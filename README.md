# tree-sitter-helium

Tree-sitter grammar for [Helium ASM](https://github.com/coffeelink/helium_asm)

## Usage

```sh
# Download
git clone https://github.com/pml68/tree-sitter-helium

# Install dependencies
pnpm install

# Build the parser
pnpm build

# Parse code
node_modules/.bin/tree-sitter parse test.hsm
```
